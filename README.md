ВОЗМОЖНОСТИ САЙТА
    Основной задачей данного сайте является показать пользователю основной спектр услуг, которые предоставляет Владельцем сайта.
    Ознакомить его с ними, и примерами готовых работ.
    Дать возможность оформить заказ, заказать обратный звонок, оставить комментарии к категориям, объектам, и оставлять отзывы.
    Для владельца, имеется возможноть большой кастомизации отображения элементов сайта и управления ими.
    Имеется система уведомлений, которая уведомит владельца о всех новых событиях на сайте.

АДМИНИСТРИРОВАНИЕ
    На главной странице администратора находится два блока.
    Первый: В этом блоке отображены все разделы (приложения) сайта. Каждый из них содержит свои настройки, методы, функции и модели, которые те в свою очередь содержат конкретные объекты.
    Втророй:    В этом блоке отображены основные элементы управления сайтом и контеном в удомбно сгруппированом виде. Этот блок разделен на три столбца.
        Первый: в нем находятся 3 элемента:
            1   -   "ДОПОЛНИТЕЛЬНО". В нем находятся две сылки. Одна (" Документация по использованию сайта") на данную документацию.
                    Вторая ("Примерная статистика") на статистику. Данный инструмент не является инструментом подсчета пользователей. А служит для примерного представления о посещаймости сайта.
                    Данная "Статистика" отобржает количество посещения пользователями главной страницы сайта.
                    Пользователь идентефицируется под данным сессии. Они обнуляются когда пользователь закрыл браузер или наступил следующий день.
                    Данные статистики за день обнуляются с наступлением следующего дня, данные за месяц соответсвенно с наступлением следующего месяца.
                    Все данные можно обнулить или изменить в разделе: "/  НАСТРОЙКИ САЙТА / НАСТРОЙКИ САЙТА (ТОЛЬКО ОДНА МОДЕЛЬ) / модель /"
            2   -   "НАСТРОЙКИ". В этом элементе сгруппированы основные элементы управления и настройки сайта.
            3   -   "КОММЕНТАРИИ И ОТЗЫВЫ". В этом элементе сгруппированы все комментарии по разделам

        Второй: в нем находится 2 элемента:
            1   -   "РАЗДЕЛ РАБОТ И УСЛУГ". В этом элементе сгруппированы все основные элементы управления контентом, относящимся к разделу "Предоставляемые услуги и все с ними связанное"
            2   -   "РАЗДЕЛ ПОРТФОЛИО". В этом элементе сгруппированы все основные элементы управления контентом, относящимся к разделу "Портфолио"

        Третий: в нем находится только 1 элемент:
            "Заказы". Этот элемент служит для быстрого доступа к имеющимся на сайте "Заказам" и "Заказам обратного звонка"
